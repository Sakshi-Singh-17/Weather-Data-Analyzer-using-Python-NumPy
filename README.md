# Weather Data Analyzer using Python and NumPy

## Project Overview
Weather Data Analyzer is a simple Python project that analyzes temperature data of a city for a month using **NumPy**.  
The program performs basic data analysis to identify temperature patterns such as hottest day, coldest day, and heatwave occurrences.

This project demonstrates **NumPy array operations, filtering, and statistical analysis**, which are important for **data preprocessing in AI/ML workflows**.

---

## Features

- Calculate the **average temperature** of the month
- Identify the **hottest day**
- Identify the **coldest day**
- Find days when temperature is **above the monthly average**
- Find days when temperature is **below the monthly average**
- Detect **heatwave days (temperature > 40°C)**

---

## Technologies Used

- Python
- NumPy

---

## Concepts Demonstrated

- NumPy arrays
- Statistical functions (`mean`)
- Index-based operations (`argmax`, `argmin`)
- Boolean filtering using `np.where`
- Data analysis using array operations

---

## Dataset

The dataset consists of **30 days of temperature values** stored in a NumPy array.

Example:

```python
temperature = np.array([
32, 34, 36, 38, 41, 39, 37, 35, 33, 30,
29, 31, 34, 36, 40, 42, 44, 43, 39, 37,
35, 33, 32, 31, 30, 34, 36, 38, 41, 45
])
```

Average temperature of the month: 35.73°C
